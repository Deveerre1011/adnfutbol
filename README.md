# ADNFutbol

App web para gestion simple de escuelas de futbol formativo en Chile y Latinoamerica.

El primer MVP se enfoca en:

- Alumnos y apoderados
- Asistencia
- Pagos
- Mensajes tipo WhatsApp
- Panel simple para profesores/directores

## Requisitos

Ya quedaron instalados en este computador:

- Node.js
- npm
- Git
- VS Code

## Comandos utiles

Entrar a la carpeta del proyecto:

```bash
cd adnfutbol
```

Instalar dependencias:

```bash
npm install
```

Abrir la app en modo desarrollo:

```bash
npm run dev
```

Revisar que el proyecto compile bien:

```bash
npm run build
```

## Carpetas principales

- `src/pantallas`: pantallas principales de la app.
- `src/componentes`: piezas reutilizables de interfaz.
- `src/datos`: datos de prueba mientras no tengamos base de datos.
- `src/tipos`: definiciones de datos como alumno, pago, grupo y asistencia.
- `src/utilidades`: funciones de ayuda.
- `docs`: documentos simples de producto y decisiones del MVP.
