# MVP ADNFutbol

## Objetivo

Lanzar rapido una herramienta simple para que una escuela de futbol pueda ordenar alumnos, asistencia, pagos y comunicacion con apoderados.

## Version 1

La primera version sera una app web responsive. Debe funcionar bien desde computador y celular.

## Funciones iniciales

- Panel principal con resumen de la escuela.
- Listado de alumnos.
- Registro de apoderados.
- Control de asistencia por grupo y fecha.
- Control de pagos mensuales.
- Mensajes prearmados para WhatsApp.

## Fuera del MVP inicial

- App nativa para App Store o Google Play.
- Portal de apoderados con cuenta propia.
- Integracion automatica oficial con WhatsApp API.
- Contabilidad avanzada.
- Gestion de torneos.
- Estadisticas deportivas complejas.

## Primer cliente ideal

Escuela pequena o mediana, con 30 a 150 alumnos, que hoy usa Excel y WhatsApp para organizarse.
